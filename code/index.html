To ensure that web sites and applications are developed in a consistent way, numerous standards have been created by W3C (World Wide Web Consortium). The standards are rules and guidelines, based on consensus of the community, that aims at making the use of the web easier, both for developers and users. By standardizing the use of web elements and languages, such as HTML and CSS, web technologies becomes consistent, regardless of the browswer or device used. The consistency also allows more efficient evolution of the world wide web since the standards constitutes a common foundation.
